Проблема?

Сейчас вся информация, представленная на сайте, выводится из базы данных. И для управления базой данных мы обращаемся к административной части сайта. Соотвественно необходимо, чтобы интерфейс административной части чайта был прост и удобен в использовании, чтобы можно было с легкостью структуризировать информацию, добавлять ее, редактировать, удалять и т.д. С помощью Django мы можем реализовать все эти функции, для удобства также можно добавить фильтры, поиск, что я и сделала. Также необходимо, чтобы сам сервис был с понятным интерфейсом и удобным, все-таки люди разных возрастов ищут для себя работу. Сайт должен иметь продуманный дизайн и функционал, и для решения этой проблемы был добавлен такой функционал как CRUD, авторизация и решистрация, вывод информации, dashboard.

Используемые технологии

Django

Фреймворк, подходящий для разработки сложных сайтов и веб-приложений, написанный на языке программирования Python. С помощью этого фреймворка осуществлялась реализация и настройка административной части, управление базой данных.

HTML

Написание самого лендинга проекта, написание статичсекого кода для главной страницы. Разметка страницы и расположение блоков.

CSS5

Написание стилей для страницы. Разработка внешнего оформления страницы и отображения элементов.

Фреймворк Materialized

Использование такого фреймворка гораздо упрощает верстку и создание адаптивного интерфейса.

JQuery

javascript библиотека, использование которой делает разработку на javascript кода намного быстрее и проще. Использовался для плавной анимации, валидации форм.

Django REST framework

Реализация RESTfull API. Работа с форматом json.
